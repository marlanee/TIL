
# Data Structure

### method (메서드)
- 객체에 속한 함수
- Data type마다 서로 다른, 고유한 메서드를 가진다.
- 메서드는 `class` 내부에 정의되는 함수다.
- 메서드 호출 방법
  - `"데이터 타입 객체.메서드()"`
  - Ex) `'hello'.capitalize()`
```python
# 문자열 메서드 예시
str_object = 'hello'
print(str_object.capitalize())  # Hello

# 리스트 메서드 예시
numbers = [1, 2, 3]
numbers.append(4)
print(numbers)   # [1, 2, 3, 4]
```

### 공통 시퀀스 메서드
- `.index(x)` : 시퀀스에서 첫 번째로 일치하는 항목 x의 인덱스를 반환
- `.count(x)` : 시퀀스에서 항목 x의 개수를 반환
```python
# index
text = 'banana'
print(text.index('a'))  # 1

my_list = [1, 2, 3]
print(my_list.index(2))  # 1

# count
text = 'banana'
print(text.count('a'))  # 3

my_list = [1, 2, 2, 3, 3, 3]
print(my_list.count(3))  # 3
```

### 탐색 및 검증
- `.find(x)` : x의 첫 번째 위치를 반환. 없으면 -1을 반환
- `.isupper(), .islower()` : 문자열의 모든 문자가 대문자 / 소문자인지 확인
- `.isalpha()` : 빈 문자열이 없고, 모든 글자가 문자일때만 True 반환. 특문, 숫자 안됨
```python
# find
text = 'banana'
print(text.find('a'))  # 1
print(text.find('z'))  # -1

# isupper
string1 = 'HELLO'
string2 = 'Hello'
print(string1.isupper())  # True
print(string2.isupper())  # False

# islower
print(string1.islower())  # False
print(string2.islower())  # False

# isalpha
string1 = 'Hello'
string2 = '123heis98576ssh'
print(string1.isalpha())  # True
print(string2.isalpha())  # False
```

### 문자열 조작 메서드 - 새로운 문자열 반환
- `.replace(old, new[,count])` : 기존 문자열의 모든 old가 new로 바뀐 문자열 반환
  - 대괄호는 선택 인자임. 안 써도 된다는 뜻
- `.strip([chars])`: 앞, 뒤의 문자를 제거함. 복사본 반환.
  - `.strip()` 처럼 괄호를 비우면 공백을 지운다.
- `split(spe=None, maxsplit=-1)`: sep을 구분의 기준으로 삼고, 문자열에 있는 단어들의 리스트를 반환.
  - 리스트를 반환하는게 핵심임
  - maxsplit이 주어지면 그 숫자 만큼만 구분함
- `.join(iterable)`: iterable의 문자열을 연결함. 복사본 반환.
  - 특이하게도, 구분자가 `''.join(iterable)` 형식임
```python
# replace
text = 'Hello, world! world world'
new_text1 = text.replace('world', 'Python')
new_text2 = text.replace('world', 'Python', 1)
print(new_text1)  # Hello, Python! Python Python
print(new_text2)  # Hello, Python! world world

# strip
# 사용자 입력 등에서 불필요한 공백이 포함된 경우
text = '   Hello    World   '

# 1. 아무것도 지정하지 않으면 '공백(띄어쓰기, 탭, 엔터)'을 제거
clean_text = text.strip()

print(clean_text)
# 결과: 'Hello    World'
# (주의: 문자열 중간의 공백은 제거되지 않음)

# 2. 제거할 문자를 지정하는 경우
text = '!!!Hello World!!!'
print(text.strip('!'))
# 결과: 'Hello World'

# [심화] 문자열 집합으로 제거 (순서 상관 없음)
# - 인자로 준 문자들을 '한 덩어리'가 아니라 '문자 집합'으로 취급
# - 양쪽 끝에서 그 집합에 속한 문자가 나오는 동안 계속 제거
# 'w', '.', 'c', 'o', 'm' 중 하나라도 양쪽 끝에 있으면 계속 제거
url = 'www.example.com'
print(url.strip('w.com'))
# 결과: 'example'
# (왼쪽의 'www.'과 오른쪽의 '.com'이 모두 제거됨)

# split
# 1. 공백을 기준으로 분리 (기본 동작)
# - 여러 개의 공백도 하나로 처리하며, 앞뒤 공백은 무시함
text = '  Hello    Python  '
print(text.split())
# 결과: ['Hello', 'Python']


# 2. 특정 문자를 기준으로 분리
# - 지정한 문자를 기준으로 '엄격하게' 분리함 (빈 문자열 발생 가능)
# - 기본 동작과 달리, 구분자가 연속되면 그 사이를 빈 문자열로 채움
data = '10,20,,30'
print(data.split(','))
# 결과: ['10', '20', '', '30']


# 3. 분할 횟수 제한 (maxsplit)
# - 앞에서부터 1번만 자르고 나머지는 그대로 둠
path = 'User/admin/documents'
print(path.split('/', maxsplit=1))
# 결과: ['User', 'admin/documents']

# join
words = ['Python', 'is', 'awesome']

sentence1 = ' '.join(words)
sentence2 = '-'.join(words)

print(sentence1)  # Python is awesome
print(sentence2)  # Python-is-awesome
```

### 리스트 값 추가 및 삭제 메서드
- `.append(x)`: 리스트 마지막에 x를 추가함
  - 복사본 반환이 아님. 원물 자체를 조작함
- `.extend(iterable)`: 리스트 마지막에 iterable을 통째로 넣음
  - iterable이 아닐 경우 추가 불가
  - 하나씩 Unpacking 해서 리스트에 넣는다.
- `.insert(i, x)`: x를 지정한 인덱스 i 위치에 삽입함
- `.remove(x)`: 리스트의 첫 번째 일치 항목을 삭제함
- `.pop() 또는 .pop(i)`: 지정한 인덱스의 항목을 제거하고 반환
  - ()로 작성하지 않을 경우, 마지막 항목을 제거하고 반환
  - *리스트에서 원하는 값을 콕 집어 뽑아오는 느낌인 듯*
- `.clear()`: 리스트의 모든 항목을 제거함. 빈 리스트로 바뀐다.
```python
# append
my_list = [1, 2, 3]
my_list.append(4)
print(my_list)  # [1, 2, 3, 4]
# append는 None을 반환합니다.
print(my_list.append(4))  # None

# extend
my_list = [1, 2, 3]
my_list.extend([4, 5, 6])
print(my_list)  # [1, 2, 3, 4, 5, 6]

# extend와 append의 비교
my_list.append([5, 6, 7])
print(my_list)  # [1, 2, 3, 4, 5, 6, [5, 6, 7]]

# my_list.extend(100)  # TypeError: 'int' object is not iterable

# insert
my_list = [1, 2, 3]
my_list.insert(1, 5)
print(my_list)  # [1, 5, 2, 3]

# remove
my_list = [1, 2, 3, 2, 2, 2]
my_list.remove(2)
print(my_list)  # [1, 3, 2, 2, 2]

# pop
my_list = [1, 2, 3, 4, 5]
item1 = my_list.pop()
item2 = my_list.pop(0)

print(item1)  # 5
print(item2)  # 1
print(my_list)  # [2, 3, 4]

# clear
my_list = [1, 2, 3]
my_list.clear()
print(my_list)  # []
```

### 정렬 및 순서 변경
- `.reverse()`: 리스트를 역순으로 변경 (정렬X)
- `.sort()`: 리스트를 오름차순으로 정렬
  - `.sort(reverse=True)` 사용시 내림차순으로 정렬됨
```python
# reverse
my_list = [1, 3, 2, 8, 1, 9]
my_list.reverse()
# reverse는 None을 반환합니다.
# print(my_list.reverse())  # None
# reverse는 원본 리스트를 변경합니다.
print(my_list)  # [9, 1, 8, 2, 3, 1]

# sort
my_list = [3, 2, 100, 1]
my_list.sort()
# sort는 None을 반환합니다.
# print(my_list.sort())  # None
# sort는 원본 리스트를 변경합니다.
print(my_list)  # [1, 2, 3, 100]

# sort(내림차순 정렬)
my_list.sort(reverse=True)
print(my_list)  # [100, 3, 2, 1]
```

## 복사
### 객체와 참조
- Mutable, Immutable: 객체 복사의 핵심 파악에 필수 / 가변, 불변
  - 가변 객체: 내용이 변경되어도 메모리 주소 동일
  - 불변 객체: 새로운 값 할당시 새로운 객체 생성
- 변수 할당의 정의: 변수는 객체의 메모리 주소를 가리키는 Label이다.

```python
# 가변(mutable) 객체 예시
print('가변(mutable) 객체 예시')
a = [1, 2, 3, 4]
b = a
b[0] = 100

print(f'a의 값: {a}')  # [100, 2, 3, 4]
print(f'b의 값: {b}')  # [100, 2, 3, 4]
print(f'a와 b가 같은 객체를 참조하는가? {a is b}')  # True

# 불변(immutable) 객체 예시
print('\n불변(immutable) 객체 예시')
a = 20
b = a
b = 10

print(f'a의 값: {a}')  # 20
print(f'b의 값: {b}')  # 10
print(a is b)  # False
```

### 얕은 복사 (Shallow Copy)
- 객체의 최상위 요소만 새로운 메모리에 복사
  - 2차원 이상을 복사하면, 2차원 값 수정은 원본에도 영향을 미침
  - 1차원만 다룰 경우 얕은 복사도 ok
- list slicing method: `b = a[:]`
- copy method: `list.copy()`
- list function: `b = list(a)`
```python
# 1. 1차원 리스트에서의 얕은 복사 (리스트 슬라이싱)
a = [1, 2, 3]
b = a[:]

print(a)  # [1, 2, 3]
print(b)  # [1, 2, 3]

print(a is b)  # False  => 별개의 객체

# 얕은 복사의 한계
print('\n다차원 리스트 얕은 복사의 한계')
a = [1, 2, [3, 4, 5]]
b = a[:]

b[0] = 999
print(a)  # [1, 2, [3, 4, 5]]
print(b)  # [999, 2, [3, 4, 5]]

b[2][1] = 100
print(a)  # [1, 2, [3, 100, 5]]
print(b)  # [999, 2, [3, 100, 5]]

print(f'a[2]와 b[2]가 같은 객체인가? {a[2] is b[2]}')  # True
```

### 깊은 복사
- 객체의 모든 수준의 요소를 새로운 메모리에 복사함
- 완전히 다른 메모리로 완벽하게 복사
- 깊은 복사 방법: copy 모듈에서 제공하는 `deepcopy()` 함수 사용
 
```python
# 복잡한 중첩 객체에서의 깊은 복사 예시
print('복잡한 중첩 객체 깊은 복사')
original = {
    'a': [1, 2, 3],
    'b': {'c': 4, 'd': [5, 6]},
}
copied = copy.deepcopy(original)

copied['a'][1] = 100
copied['b']['d'][0] = 500

print(f'원본: {original}')  # {'a': [1, 2, 3], 'b': {'c': 4, 'd': [5, 6]}}
print(f'복사본: {copied}')  # {'a': [1, 100, 3], 'b': {'c': 4, 'd': [500, 6]}}
print(
    f"original['b']와 copied['b']가 같은 객체인가? {original['b'] is copied['b']}"
)  # False
```

### List Comprehension
- 리스트 생성 방법임: 간결하고 효율적
- 단, 구조가 복잡할 경우 일반적인 for문을 사용하자
- *구조가 낯설어서 익숙해질 필요가 있다.*
- List Comprehension 구조: `list = [expression for 변수 in iterable if 조건식]`

```python
# List Comprehension 활용 예시   # 인셉션이냐
# "2차원 배열 생성 시 (인접행렬 생성 시)"
data1 = [[0] * (5) for _ in range(5)]
print(data1)
# 또는
data2 = [[0 for _ in range(5)] for _ in range(5)]
print(data2)
```

### 리스트를 생성하는 3가지 방법
- loop
- list comprehension: 가장 pythonic, 대부분 우수한 성능
- map

```python
# 1. loop
result1 = []
for i in range(10):
    result1.append(i)

# 2. list comprehension
result2 = [i for i in range(10)]
# result2 = list(i for i in range(10))

# 3. map
result3 = list(map(lambda i: i, range(10)))
```
