# 자릿수 더하기
최대 4자리의 자연수 하나를 받아서 각 자릿수를 더한 값을 출력

## 풀이 코드
```python
N = list(map(int, input()))
T = 0

for i in N:
    T += i

print(T)
```

### 아쉬운 점
- 처음에 코드 짤 때 오류가 발생함
    - N = `map(lsit(int, input()))` 으로 코드를 짰기 때문. 
    - 작성한 코드의 문법이 정확한지 오류 발생시 확인 필요

### Feedback Code
```python
# 가장 파이썬다운(Pythonic) 방식
total = sum(map(int, input()))

print(total)
```
### Gemini Feedback
- memory 사용량을 늘리는 list() 함수를 쓸 필요가 없음.
    - map() 함수만 사용해도 반복 가능한 객체 (Iterable)이 나온다.
        - 한 번 쓰면 사라지는 숫자 자판기.
        - 정렬, 추가, 길이 확인 등의 기능은 없기에 사용 주의 필요.
- 파이썬 내장 함수 sum() 사용. 실무에서 사용하는, 가독성과 성능 좋은 함수임.

## 총평
